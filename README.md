# Snake Game - The Ball Can Run

## Link
https://thienquan-art.github.io/Snake-Game---The-Ball-Can-Run/

## Game Description
The Prey That Runs is a Snake game with a more active chase. Instead of collecting food that stays in one place, you hunt a quick red ball that tries to get away. Each time you catch it, the arena gets smaller, so the game gets tighter, faster, and harder to survive.

## Features

### Smart Prey
The red ball reacts to you. When you get close, it runs. If you keep pressuring it, it can use a one-time long sprint to escape to a far corner.

### Shrinking Arena
Every catch makes the board smaller. That means less room to move and more pressure as the game goes on.

### Sprint Boost
Press the spacebar to give your snake a short speed burst. You can use it anytime, so timing it well makes a big difference.

## How to Play

### Objective
Control your snake and catch the red ball.

### Controls
Arrow Keys or W, A, S, D — Move up, down, left, or right
Spacebar — Activate a temporary speed boost

A simple way to think about it: it’s Snake, but instead of eating food, you’re chasing prey that fights to stay alive.
